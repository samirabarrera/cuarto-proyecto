# 💰 Aplicación de Control de Gastos

Una aplicación web completa y eficiente para la gestión de finanzas personales. Permite a los usuarios registrar ingresos, clasificar gastos en tiempo real y mantener un control estricto de sus presupuestos mensuales.

---

## 🚀 Características Principales

* **Registro de Transacciones:** Añade ingresos y gastos con descripción, monto, fecha y categoría.
* **Categorización Personalizada:** Clasifica los movimientos (Comida, Transporte, Servicios, etc.).
* **Dashboard Visual:** Gráficos e historiales para entender el flujo del dinero.
* **Historial Completo:** Filtra y busca transacciones pasadas fácilmente.

---

## 🛠️ Tecnologías Utilizadas

### Frontend
* **React (con Vite):** Entorno de desarrollo rápido y biblioteca para construir la interfaz de usuario.
* **Bootstrap:** Framework de CSS para un diseño limpio, responsivo y ágil.

### Backend
* **Node.js & Express:** Entorno de ejecución y framework para construir una API REST robusta y escalable.
* **Auth0 Node SDK:** Integración y validación de tokens JWT en el backend para proteger las rutas de la API.

### Base de Datos
* **PostgreSQL:** Sistema de gestión de bases de datos relacionales para almacenar usuarios y transacciones de forma segura.
